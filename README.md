# Gatsby Coffee
This is a coffee online shopping mall.
https://gatsby-coffee-olivia.netlify.com

## Technology Stack
* Front-end: React, Bootstrap4, HTML, CSS
* Back-end: Contentful(CMSs)
* Data Query: GraphQL
* Web Deploy: Netfily

## Project Module
### menu module
* goods display
* type filter

### product module
* product display

### shopping cart module
* add product in cart
* modify product amount
* checkout

### contact module
* email sending
